Welcome to FriendPerts

FriendPert Page Explanations

Asks

Asks are questions that the user poises for their FriendPerts. Users associate tags with their questions and Answerers that are associated with those tags are notified of the user's Ask.

Answers

The Answers page contains all of the questions that a user's FriendPerts have asked them. This page is where users can answer their FriendPerts' questions.

Answerers

The Answerers page contains a list of all available FriendPerts and allows a user to add them to their personal Answerer's list. You add answerers from the "Friendslist" page, which is populated with all your FB friends.
